🏦 How Direct Salary Disbursement Works

📋
Payroll Computed
Net pay calculated per employee after all statutory deductions
✅
Authorise Run
Employer approves batch; funds debited from company account
🔀
API Call
CPS platform calls bank Payouts API (RazorpayX / Cashfree / YES Bank)
⚡
IMPS / NEFT
Funds transferred to each employee's bank account in real time
📨
UTR + Slip
Each employee gets UTR reference & salary slip via email/SMS

💡 How Competing Payroll Platforms Handle Direct Disbursement

Razorpay Payroll
Uses RazorpayX Payouts API — IMPS/NEFT batch with real-time status webhook. Native banking rails, no separate bank integration needed.
Limit: ₹2L per txn via IMPS; NEFT for higher amounts
Greythr / Keka
Generates bank-specific payment files (SBI, HDFC, ICICI, Axis formats). Employer uploads to net banking portal manually. No real-time status.
Manual upload step required; no API-based transfer
Zoho Payroll
Integrates with Zoho Books / ICICI Bank directly. Auto-debit from linked current account on salary date. Email confirmation sent.
Only ICICI Bank direct debit supported natively
Cashfree Payouts
Batch Payouts API with IMPS/NEFT/UPI. Per-transaction webhook, instant UTR. Used by several HR SaaS providers as white-label.
Best-in-class API; UPI support up to ₹1L
Darwinbox
Bank-agnostic file generation + direct integration with Yes Bank CMS and HDFC SmartPay for enterprise clients.
Enterprise-only; requires bank partnership setup
CPS LLP Copilot
Simulated below — in production connects to RazorpayX / Cashfree Payouts API. Real-time per-employee status stream with UTR generation.
✓ API-ready · Any bank · Real-time
